## üîÑ How to Convert SVG to PNG

### **Option 1: Online Converter (Easiest & Fastest)**
1. Go to [CloudConvert](https://cloudconvert.com/svg-to-png) or [Convertio](https://convertio.co/svg-png/)
2. Upload `exported-logos/logo-b-app-store-1024x1024.svg`
3. Set dimensions to **512x512 pixels** (or keep 1024x1024 - Google accepts both)
4. Make sure **"Remove transparency"** or **"Background color: White"** is checked
5. Download the PNG file

### **Option 2: Using Inkscape (Free, Recommended)**
1. Download [Inkscape](https://inkscape.org/) (free)
2. Open `exported-logos/logo-b-app-store-1024x1024.svg`
3. Go to **File ‚Üí Export PNG Image**
4. Set size to **512 x 512 px** (or 1024x1024)
5. Click **Export**
6. The PNG will be saved

### **Option 3: Using Browser + Screenshot**
1. Open `exported-logos/logo-b-app-store-1024x1024.svg` in Chrome/Firefox
2. Take a screenshot or use browser dev tools
3. Crop to 512x512 or 1024x1024
4. Save as PNG

### **Option 4: Using Node.js Script**
If you have Node.js installed, you can use a script (see below).
