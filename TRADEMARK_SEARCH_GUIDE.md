**Website**: https://www.uspto.gov/trademarks/search

**Steps**:
1. Go to the USPTO Trademark Electronic Search System (TESS)
2. Select "Basic Word Mark Search (New User)"
3. Search for: `Balanze`
4. Also search for: `BALANZE` (all caps)
5. Check for similar names: `Balance`, `Balanze App`, `Balanze Finance`

**What to Look For**:
- ‚úÖ **No results** = Good (name likely available)
- ‚ö†Ô∏è **Inactive/Abandoned** = Usually okay, but check status
